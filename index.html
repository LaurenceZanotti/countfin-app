Como o projeto nasceu

Por causa de duas necessidades básicas, a de ter um controle financeiro das contas e de aprender JavaScript, nasceu a idéia de fazer um aplicativo web de controle de contas.

Tive essa idéia enquanto assistia um tutorial que ensinava a fazer um app que guardava uma lista de livros, e decidi usar esse vídeo como base para o meu projeto.

Com o tempo, enquanto entendia os conceitos básicos do JavaScript e do DOM, fui buscando implementar mais funcionalidades originais, como a busca, edição da conta, e o dashboard, com uma visão geral das contas do mês.

O que falta melhorar

Tenho intenção de deixar este app mais robusto com o tempo, reescrevendo o código e implementando novas funcionalidades. Toda sugestão e feedback são bem vindos. Se quiser dar sua opinião e/ou me contatar, por favor, sinta-se livre para usar um dos links desta seção ou por e-mail!

No futuro devo adicionar:

  • Back end (talvez)
  • Criptografia
  • Salário e outras entradas de renda
